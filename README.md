# unity_QuizGame_Template
Hi everyone, this is version 1 of this simple quiz game made in unity 6000.2.6f2
Basic quiz framework for beginners, takes a total of 6 questions per round, 4 options each questions 
System can dynamically load and change questions in the build file via StreamingAssets folder
